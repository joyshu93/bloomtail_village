# 3D Vertical Slice Scope

This branch is a narrow vertical slice, not the full cozy village MVP.

Included:

- Flat 3D grid floor
- Camera3D movement and wheel zoom
- Mouse hover cell detection via camera-to-ground hit
- Placeable selection UI
- Ghost preview with valid / invalid feedback
- Click placement with grid snap
- Four placeables: Road, House, Cafe, Tree
- Road-adjacent active / inactive state for House and Cafe

Excluded:

- Residents
- Save/load
- Economy
- Quests
- Seasons
- Sound
- Final art
- Pathfinding
